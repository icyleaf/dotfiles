## Agent skills

### Issue tracker

Issues and PRDs for this repo live as GitHub issues. See `docs/agents/issue-tracker.md`.

### Triage labels

The default triage labels are used. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout with a root `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.

### Language

All conversational responses to the user MUST be written in **Simplified Chinese (中文)**. This applies to explanations, questions, summaries, and any prose directed at the user.

All written artifacts MUST remain in **English**: source code, comments, commit messages, GitHub issues, specs, ADRs, `CONTEXT.md`, `README.md`, and any other file committed to the repository.
